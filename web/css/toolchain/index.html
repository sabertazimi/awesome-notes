Normalize​

Minimal:

html {
  -webkit-font-smoothing: antialiased;
  scroll-behavior: smooth;
  text-wrap: pretty;
}

::selection,
::target-text {
  color: #fff;
  background: #000;
}

@view-transition {
  navigation: auto;
}
