# Algorithms

## Algorithm Topics

- **[Sorting](./sorting.md)** - 排序算法：Selection Sort, Insertion Sort, Shell Sort,
  Merge Sort, Quick Sort, Heap Sort, Radix Sort
- **[Tree](./tree.md)** - 树算法：Binary Search Tree, 2-3 Tree, Red-Black BST, B Tree,
  Fibonacci Heap, K-D Tree, Tree Edit Distance
- **[Graph](./graph.md)** - 图算法：图的表示, BFS, DFS, 拓扑排序, 最短路径, 最大流, 最小生成树, 强连通分量
- **[Search](./search.md)** - 搜索算法：First Search, Binary Search, Range Query, Asymmetry of Verification
- **[Dynamic Programming](./dp.md)** - 动态规划：子问题, 基础 DP 问题, 数位 DP
- **[Greedy](./greedy.md)** - 贪心算法：贪心选择, Huffman Tree
- **[String](./string.md)** - 字符串算法：KMP Algorithm, Rotate String, Palindrome String
- **[Data Structure](./data-structure.md)** - 数据结构：Array, List, Stack, Map, Set, BitMap
- **[Math](./math.md)** - 数学算法：Radix, Matrix Fast Power, Mod Power, XOR
- **[Simulation](./simulation.md)** - 模拟算法：Zig Zag, Matrix Traversal, Painting, Meet and Collision
