# Layout

## Section

必须含有**hx**标题子标签.

## Header

Not only can the page `<body>` contain a header,
but also can every `<article>` and `<section>` element.

## Footer

Not only can the page `<body>` contain a footer,
but also can every `<article>` and `<section>` element.

## Nav

- 传统导航条
- 侧边栏导航
- 页内跳转
- 翻页操作

## Main

- 每个网页只有 1 个`main`元素
- `main`不可为`article`、`aside`、`header`、`footer`、`nav`孩子
