# Visual Mode

可视模式:

| 命令      | 作用                                    |
| :-------- | :-------------------------------------- |
| `v`       | 激活面向字符的可视模式                  |
| `V`       | 激活面向行的可视模式                    |
| `<C-v>`   | 激活面向列块的可视模式                  |
| `gv`      | 重选上次的高亮选区                      |
| `shift-v` | select current line                     |
| `gq`      | split selected line into multiple lines |
| `o`       | 切换高亮选区的活动端                    |
