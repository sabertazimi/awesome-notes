Data Monoid#

  • 成员: 必须为具体类型(*不可是类型构造符(type constructor))
  • 准则(Monoid Law):
    • 结合律 a·(b·c) = (a·b)·c
    • 无需满足 a mappend b == b mappend a
class Monoid m where    mempty :: m             -- identity    mappend :: m -> m -> m    mconcat :: [m] -> m    mconcat = foldr mappend mempty
