Macro

命令作用
q{register}开始录制宏,并将其保存到指定寄存器,再次按 q 结束录制
{number}@{register}命令执行指定寄存器的内容
@@重复 @ 命令

register 为小写时,覆盖录制宏; register 为大写时, 追加录制宏.
